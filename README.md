# Estudos em Java

Este repositório contém alguns estudos e experimentos feitos durante o aprendizado da linguagem **Java**. A ideia é usar este espaço para praticar conceitos, testar códigos e registrar a evolução no uso da linguagem.

Os arquivos aqui podem incluir exercícios simples, testes de funcionalidades da linguagem e pequenos exemplos voltados para aprendizado.

O objetivo principal é consolidar conhecimentos em Java por meio da prática contínua.
